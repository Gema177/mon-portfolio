HTML et CSS

HTML est le langage de balisage utilisé pour créer des pages web. Il est à la base de toute structure de page web.

CSS est utilisé pour styliser les pages web en définissant l'apparence et la mise en page des éléments HTML.

  • HTML permet de structurer le contenu d’une page web.
  • Les balises de structure de base en HTML incluent <html>, <head>, et <body>.
  • CSS permet de définir des règles de style sur des éléments HTML.
  • Flexbox et Grid sont des outils puissants en CSS pour créer des mises en page responsives.
  • Les media queries permettent de rendre le site adaptatif.
<html>...</html> <head>...</head> <body>...</body>
JavaScript

JavaScript est un langage de programmation utilisé pour créer des pages web interactives.

Il permet d'ajouter des fonctionnalités dynamiques aux pages web comme les animations, les validations de formulaire, etc.

  • JavaScript permet de manipuler le DOM et d'interagir avec les éléments HTML d'une page.
  • Les variables en JavaScript peuvent être déclarées avec les mots-clés `let`, `const`, ou `var`.
  • Les fonctions sont utilisées pour encapsuler des morceaux de code réutilisables.
  • Les événements JavaScript réagissent aux actions des utilisateurs comme un clic ou un survol.
  • Les modules ES6 permettent de diviser le code en parties réutilisables et organisées.
console.log("Hello, World!"); let x = 10; function myFunction() { ... }
Python

Python est un langage de programmation polyvalent, utilisé dans de nombreux domaines comme le web, la science des données, et l'intelligence artificielle.

La syntaxe de Python est simple et intuitive, ce qui le rend facile à apprendre pour les débutants.

  • Les variables en Python n'ont pas besoin d'être déclarées avec un type spécifique.
  • Les fonctions en Python sont définies avec le mot-clé `def`.
  • Les boucles `for` et `while` sont couramment utilisées en Python.
  • Les dictionnaires sont utilisés pour stocker des paires clé-valeur.
  • Les exceptions sont gérées avec les blocs `try` et `except`.
def my_function(): return "Hello, Python!" x = 10
SQL

SQL est un langage standard utilisé pour interroger et manipuler des bases de données relationnelles.

Les requêtes SELECT sont utilisées pour extraire des données d'une base de données.

  • Les clauses WHERE et JOIN permettent de filtrer et combiner des données de plusieurs tables.
  • Les commandes INSERT INTO et UPDATE sont utilisées pour ajouter ou modifier des données.
  • Les transactions SQL permettent de garantir la cohérence des données en cas d'erreur.
  • Les fonctions d'agrégation comme COUNT, SUM, et AVG sont utilisées pour calculer des résultats.
  • Les sous-requêtes permettent d'effectuer des requêtes imbriquées.
SELECT * FROM users; INSERT INTO users VALUES ('Alice', 30); UPDATE users SET age = 31 WHERE name = 'Alice';
Git

Git est un système de gestion de version décentralisé, utilisé pour suivre les changements dans les fichiers de code source.

Git permet de collaborer efficacement sur des projets de développement de logiciels avec plusieurs développeurs.

  • Les commandes Git de base incluent `git init`, `git add`, `git commit`, et `git push`.
  • Les branches Git permettent de travailler sur différentes versions sans interférer avec la branche principale.
  • Les commandes `git merge` et `git rebase` sont utilisées pour intégrer les changements d'une branche dans une autre.
  • GitHub est une plateforme qui permet d'héberger des projets Git et de collaborer en ligne.
  • Les conflits de fusion peuvent survenir lorsqu'il y a des modifications concurrentes dans un fichier, et doivent être résolus manuellement.
git init git commit -m "Message" git push origin main
