Description du Projet

Ce projet innovant combine hardware et software pour créer une solution d'incubation intelligente. La couveuse est équipée de capteurs de pointe et d'un système de contrôle automatisé, le tout piloté par une application mobile intuitive.

Technologies Utilisées

  • Arduino/ESP32 pour le contrôle
  • Capteurs de température et d'humidité DHT22
  • Application Android native
  • Intelligence artificielle pour l'optimisation
  • Communication en temps réel via WiFi
